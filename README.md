# Trabajo_Final_NYC_Taxis
Trabajo final del curso Analítica de los negocios. Incluye los datos utilizados, script en R, visualizaciones generadas y el informe final, organizados para documentar de manera estructurada todo el proceso de análisis exploratorio, modelación econométrica de la demanda de taxis amarillos en Nueva York 2024 y los resultados obtenidos.
